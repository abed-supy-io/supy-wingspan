## supy-ai-agents — polyglot AI-agents monorepo (Node.js + Python + Cloudflare Workers) — candidate NEW stack: ai-agents

- **Purpose:** Autonomous AI agents + knowledge-graph platform — cortex (KG extractor/validator + MCP), nexus (sales intelligence), pmsai (PMS vision), oculus (dashboard), gleap-dashboard (support), pos-integration.
- **Structure:** 11 independent packages + git submodule (supy-onboarding); NO root workspace orchestration. Cortex (Node 22, Express + BullMQ), Nexus (Python Poetry + FastAPI), Oculus (Turbo monorepo), PMS-AI (Python FastAPI), Gleap (Cloudflare Pages). Each self-contained.
- **Architecture & patterns:** Cortex = Claude Agent SDK + MCP server (19 tools over SSE/HTTP), Express + BullMQ (Redis job queue, cluster support), PostgreSQL + pgvector KG, repo-clone→extract→inject pipeline. Nexus = FastAPI + MCP brain-graph queries + managed-agents. Oculus = Turbo/npm workspaces + TS 5.7. Gleap = CF Pages + OAuth 2.1. Errors via try/catch + custom loggers, rate-limit guards, payload validation.
- **Tooling:** lint=(eslint/ruff inferred) · format=(prettier/black inferred) · test=(node --test for Cortex; pytest Nexus; Turbo test Oculus; none PMS-AI) · CI=GitHub Actions (daily POS report, testo AI test-gen) · codegen=custom extraction pipeline + Turbo generators · pre-commit=native git hook (deploy/ secret scanning) · commits=feat/fix/docs/chore/refactor (manual semver) · deploy=Cortex podman build linux/amd64 → Artifact Registry → manifest bump → ArgoCD; Gleap `wrangler pages deploy`; Nexus K8s Jobs.
- **Testing:** node native runner (limited — 4 tests in Cortex); pytest (Nexus); Turbo (Oculus); PMS-AI run_tests.py; no mocking framework evident.
- **Security / secrets / config:** wrangler secrets (GLEAP_API_KEY, OPENROUTER_KEY, OAuth client/secret, SLACK_WEBHOOK); env for DB/Redis/GitHub/Anthropic keys; .env.example templates; OAuth 2.1 + JWT signing (cortex/oauth-server.js); GitHub App auth; Bearer API keys; MCP auth token (read-only engineer access); Cortex Docker runs non-root (UID 1001). — do not reproduce any secret value.
- **Divergences vs Supy TS conventions:** no root Nx/Turbo orchestration; ES modules; Claude Agent SDK + MCP; BullMQ (not NATS); Postgres+pgvector (not Mongo); polyglot Node+Python; Cloudflare Pages (not K8s) for Gleap.
- **New patterns worth codifying:** MCP server pattern (tool/resource mounting + OAuth 2.1/JWT auth); extract+inject KG pipeline; Redis Cluster config with fallback; env-driven config singleton; custom HTTP MCP auth; logger with time-tracking.
- **Recommendation:** **candidate NEW stack (ai-agents)** — authoritative pattern for autonomous agents + KG + MCP. Codify as a distinct profile (BullMQ, MCP server, extraction pipeline, pgvector, OAuth 2.1/JWT, CF Pages, podman linux/amd64, Node+Python polyglot).
