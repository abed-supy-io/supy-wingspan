## supy-frontend — Angular 21 + Nx 22.4 + NGXS + PrimeNG (angular-nx)

- **Purpose:** Multi-tenant B2B admin/retailer/supplier platform — role-based access, data grids, forms, real-time WebSocket features.
- **Structure:** `apps/{admin,retailer,supplier}` + ~40 domain-scoped `libs/` (authentication, catalog, orders, inventory, items, channels…) with `type:app|lib` + scope tags; Nx monorepo + npm; lazy routes via `provideStates([...])`.
- **Architecture & patterns:** `ChangeDetectionStrategy.OnPush` mandatory + `inject()` DI. NGXS store: action classes + `EntityListState` base (common lib) for CRUD lists; Immer `produce()` for immutable draft updates; `@Selector()` + `selectSignal()`. Signals throughout (486 uses) + `toSignal()` interop. `BaseHttpService` wraps HttpClient with URI tokens. Filter actions accept `{ saveToUrl: false }` in dialogs/drawers to avoid navigation-close race. Typed reactive forms. Scoped SCSS with `--p-*` PrimeNG token overrides (no `::ng-deep`). Lazy routing with nested `children[]` guards.
- **Tooling:** lint=ESLint 9.36 (@angular-eslint, simple-import-sort; relative/`path/index` imports banned) · format=Prettier 3.8.1 (120 cols, single quotes) · test=vitest-angular + jest-preset-angular (coverage json-summary/lcov/text; excludes .spec/.stories/.module/index) · CI=GitHub Actions preview.yml (branch preview URLs) · codegen=@nx/angular generators (OnPush + SCSS default) + Storybook 10.3.3 · pre-commit=Husky + lint-staged (eslint --cache --fix, stylelint --fix, prettier --write) · commits=@supy/commitlint-config (feat|bug|chore).
- **Testing:** vitest-angular + jest-preset; `buildTarget: components:build`; Jasmine-style describe/it; ci config enables coverage, parallel=3; no snapshot testing.
- **Security / secrets / config:** dotenv-vault multi-env (.env{,.development,.production}) per app; env:pull/push via GCloud; `environment.json` manifest + `APP_CONFIG` InjectionToken; hidden source maps in retailer prod; no hardcoded secrets; GCloud auth required.
- **Divergences vs a typical Supy angular-nx repo (vs mined starter-kit conventions):** matches exactly — OnPush mandatory, inject() DI, NGXS + Immer, BaseHttpService, lazy routing, PrimeNG preset styling, no `::ng-deep`.
- **New patterns worth codifying:** NGXS `EntityListState` + Immer `produce()` immutability; `saveToUrl:false` filter-action pattern in modals; `selectSignal()` + `computed()` reactivity; `AppConfig` token + env-driven bootstrap; Storybook-native component library (libs/components); coverage exclusions for .stories/.module/grids.
- **Recommendation:** deepen angular-nx — mature, rigorously enforced. Codify: mandatory OnPush + inject(), Immer in NGXS, saveToUrl pattern, PrimeNG preset overrides (no `::ng-deep`), signal reactivity, BaseHttpService URI tokens, lint rules (no relative imports, no any, const+readonly, 50-line fns, 5-param max). Candidate: PrimeNG styling guide (preset composition + `--p-*` token overrides).
